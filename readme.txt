你说得对，但是《悬浮时钟》是由宋浩然开发的一款时钟显示软件，你需要双击“main.exe”打开...（前面没忘，中间没忘，后面忘了）

==================================
点击目录中的"main.exe"运行主程序，此时打开的是时钟启动器，可以修改配置，然后点击右下角的三角形按钮打开时钟，启动器会最小化到任务栏，如需修改配置可以
点击时钟右上角的时钟图标，或在任务栏打开

2.9.8及以上的更新说明
如果在更多页面里启用"下次启动时,直接打开时钟"，会直接打开时钟，不显示配置界面
如果启用"下次启动时,使用我上次的设置",在下次打开时会恢复到上次的配置
建议配置好后点一下右下角应用按钮，然后再开启"下次启动时,使用我上次的设置"

==============问答=================
怎么回事时钟关不掉
|点两下关闭键即可关闭，如仍无法关闭，可以直接在任务栏直接关闭时钟启动器，如仍旧无法关闭，可尝试使用任务管理器结束进程

修改完配置，但是未生效
|建议配置好后点一下右下角应用按钮，使得更改应用到时钟
